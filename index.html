CompTIA NETWORK+ Guide to Network

Learning a new thing on a technical field is challenging, especially when you have to do it in such a short period of time. The outcome of this class excedeed my expectations because having an empathic professor helped a lot to release the pressure of

going back to school after many years. Having a young professor with the willingness to share his knowledge and professional experience during class, I was able to get a hint on what is the basic level of expertise expected on the professional IT world nowadays.

This class was not only base on the syllabus but instead was a resource full of practical and real escenarios that allowed to analize the subject from a real pespective.

Some of the subjects that this class included are :

  • Networking Concepts
    • How computers find each other on networks and how data is transported.
    • Structured Cabling and Wireless Networking
    • Network risks and management
    • Quizzes to be prepare to pass CompTIA NETWORK+ N10-006 certification axam.
  • Introduction to different programming languages
    • Explanation of pros and cons among different languages:Assembly,C,C++,Java, C#, Python
    • The roadmap to becoming a Web developer in 2018
    • HTML and its practical demostration, developing a Resume web page.
    • Python and some scripts demostrations.
  • Resource of helpful tools and websites
    • Network snifter: Wireshark
    • Programming Simulation site:Repl.it
    • Spycloud
    • Informationisbeautiful.net
    • Visual Studio 2017
    • Free web page hosting:Github
    • Text editor:Notepad++
    • Learn how to code:Code Academy

It is a fact that it takes effort and more than one class to advance in the field but this was definitely a great start to explore the broad range of options than this career has to offer.
